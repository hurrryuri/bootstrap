input태그는 키보드로 값을 입력받아 서버 및 스크립트에 값을 전달

        [형식]
        

        [type 종류]
        text
        password
        tel
        number(min, max)
        range(min, max)
        date
        time
        datetime-local
        month
        file
        checkbox(name을 다르게)
        radio(name은 동일하고 값은 다르게)
        submit
        reset
        button
        search
        color

        Form은 input의 name(변수)를 그룹화해서 전달(DTO 또는 개별변수로)
        


        input태그에 제목을 담당

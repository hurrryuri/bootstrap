해당 요소를 기준으로 해서 바깥쪽 여백(margin), 안쪽 여백(pading)으로 구성
        크기는 1~5 사이의 값

        [바깥쪽여백]
        m-크기 : 상하좌우의 여백을 적용
        m-t크기 : 위쪽 여백을 적용(Top)
        mb-크기 : 아래쪽 여백을 적용(Bottom)
        ml-크기 : 왼쪽 여백을 적용(left)
        mr-크기 : 오른쪽 여백을 적용(Right)
        mx-크기 : 좌우여백을 적용
        my-크기 : 상하여백을 적용

        [안쪽쪽여백]
        p-크기 : 상하좌우의 여백을 적용
        p-t크기 : 위쪽 여백을 적용(Top)
        pb-크기 : 아래쪽 여백을 적용(Bottom)
        pl-크기 : 왼쪽 여백을 적용(left)
        pr-크기 : 오른쪽 여백을 적용(Right)
        px-크기 : 좌우여백을 적용
        py-크기 : 상하여백을 적용

        각 요소별 간격 또는 크기지정
        여백시 내용이 가운데 존재하면 여백이 적용
        여백시 내용이 상단에 존재하면 여백이 적용되지 않은 상태

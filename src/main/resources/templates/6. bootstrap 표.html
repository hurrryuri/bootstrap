    [표 모양]
    .table .table-bordered : 테두리가 있는 표
    .table .table-striped : 무늬가 있는 표
    .table .table-hover : 마우스에 반응하는 표
    .table .table-dark : 어두운 색의 표
    .table .table-자비(sm,md,lg,xxl) : 크기
    .table .table-responsive : 반응형 표

    CSS구조
    태그명 {속성(색상, 글꼴, 배경색, 위치)} : 해당하는 모든 태그에 적용
    #아이디명 {속성(색상, 글꼴, 배경색, 위치)} : 하나의 태그에 적용(페이지에 단 한번만)
    .클래스명 {속성(색상, 글꼴, 배경색, 위치)} : 지정된 모든 태그에 적용
